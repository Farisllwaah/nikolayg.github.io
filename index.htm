Hi there,

My name is Nikolay Grozev. I'm a PhD candidate at the Cloud Computing and Distributed Systems (CLOUDS) Laboratory at the University of Melbourne, Australia. My supervisor is Professor Rajkumar Buyya. Currently I'm working on Cloud technologies, but my broader interests include Software Engineering, Software Architectures, Quality Assurance and Machine Learning.

In my PhD I investigate how to architect 3-tier software applications so that they can utilise multiple Cloud data centres in order to be legislation compliant, more resilient to data centre outages and to increase users’ Quality of Experience (QoE). I am developing novel formal models, adaptive software architectures and am using online machine learning techniques to “learn” the application’s resource requirements in real time and allocate computing resources adequately.

This is an exciting and important topic and you can learn more by exploring my latest publications.
